# BOSTON_HOUSE_PRED

### Software and tools Requirements

1. [Github Account](https://github.com)
2. [Visual_Studio_Code_IDE](https://code.visualstudio.com/)
3. [Heroku_Account](https://heroku.com)
4. [Git_CLI](https://https://git-scm.com/install/windows)

Create a new Environment

```
conda create -p venv python==3.10.9 -y
```

Activate Environment

```
conda activate venv/
```
